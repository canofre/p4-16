# Multiplicação e divsão de reais

# Introdução

Baseado no exemplo de calculadora e utilizando o formato

             0                1                  2              3
      +----------------+----------------+----------------+---------------+
      |      P         |       4        |     Version    |     Op        |
      +----------------+----------------+----------------+---------------+
      |                              Operand A                           |
      +----------------+----------------+----------------+---------------+
      |                              Operand B                           |
      +----------------+----------------+----------------+---------------+
      |                              Result                              |
      +----------------+----------------+----------------+---------------+
 

-  P is an ASCII Letter 'P' (0x50)
